# COVID-19 Data Analysis Project

A comprehensive analysis of COVID-19 data across different regions and time periods, visualizing trends and patterns to provide insights into the pandemic's progression.

## Personal Experience

Working on this project during the height of the pandemic gave me a deeper understanding of how data science can be applied to real-world problems with immediate impact. The most challenging aspect was handling inconsistent data reporting across different regions and time periods. I developed robust data cleaning processes that could accommodate these inconsistencies while still producing reliable insights.

I learned a great deal about time-series analysis and geospatial data visualization, skills that I've since applied to many other projects. This project also reinforced the importance of responsible data presentation, especially when dealing with sensitive public health information.

## Technical Overview

This project encompasses:

- Data collection via APIs from Johns Hopkins University and other public health sources
- Extensive data cleaning and preprocessing with Pandas
- Time-series analysis of case, recovery, and mortality rates
- Interactive visualizations using Plotly and Dash
- Predictive modeling using Prophet and scikit-learn
- A web dashboard for exploring the data interactively

The codebase is structured as a reusable package that can be adapted for other epidemiological data analysis tasks.
